
# Log Colors

It can color your like console.success("Hello World!!!");


## Installation

Install log-colors with npm

```bash
  npm install log-colors
```
    
## Usage/Examples

```javascript
import Log from 'log-colors'
console.success("Hello World"); // It show "Hello World!!!" in green color in console.
```
